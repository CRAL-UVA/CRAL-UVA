# Traxxas Robot Documentation

Welcome to the **Traxxas UDR Robot** documentation for CRAL-UVA!

The Traxxas Unlimited Desert Racer (UDR) is a high-performance RC platform modified for robotics research. This documentation covers setup, operation, and maintenance for the Traxxas vehicles at CRAL.

## Platform Specifications

- **Base Vehicle:** Traxxas Unlimited Desert Racer (UDR)
- **Compute Unit:** NVIDIA Jetson Xavier NX
- **Sensors:** Hokuyo 10LX LiDAR, Intel RealSense D435i
- **Motor Controller:** VESC
