# Jackal Robot Documentation

Welcome to the **Clearpath Jackal Robot** documentation for CRAL-UVA!

The Clearpath Jackal is a rugged, lightweight, fast, and easy-to-use unmanned ground robot designed for rapid prototyping and research applications. This documentation provides comprehensive guides for setup, operation, and maintenance of the Jackal robots at CRAL.

## Getting Started

New to the Jackal robots? Start here:

1. **First Time Setup:** Begin with the [ROS Installation Guide](Setup_files/Ros_installation.md) to set up your development environment
2. **Sensor Configuration:** Configure your sensors using the [Sensor Drivers Guide](Setup_files/sensor_drivers.md)
3. **Basic Operation:** Learn how to operate the robot with the [User Manual](Main_files/User_Manual.md)
4. **Tutorials:** Follow step-by-step [Tutorials](Main_files/Tutorials.md) for common tasks
